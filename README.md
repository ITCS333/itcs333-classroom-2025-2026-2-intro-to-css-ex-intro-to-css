# CSS Styling Exercise

Welcome! In this assignment you will practice writing CSS by styling a pre-built HTML page.
You will **only** edit one file: `styles.css`. The HTML file is provided for you and
should not be modified.

---

## Project Structure

```
css-exercise/
├── index.html        ← Provided. DO NOT modify this file.
├── styles.css        ← Your work goes here.
└── tests/
    ├── styles.test.js  ← Provided. DO NOT modify this file.
    └── package.json
```

---

## Your Task

Open `styles.css`. You will find **15 comment blocks**, each describing exactly what
CSS rules to write. Read each comment carefully and write the correct CSS beneath it.

The exercises cover:

| # | Topic |
|---|---|
| 1 | Global body styles — font, background, color, reset |
| 2 | Header background, color, padding, and alignment |
| 3 | Header `<h1>` font size and margin |
| 4 | Navigation link color, decoration, spacing, and size |
| 5 | Navigation link hover state |
| 6 | Main content area max-width and centering |
| 7 | Section heading color and bottom border |
| 8 | Intro paragraph font size, line height, and style |
| 9 | Hobbies list reset and list item styling |
| 10 | Highlighted list item override |
| 11 | Blockquote background, padding, and border |
| 12 | Cite element display, alignment, and color |
| 13 | Button link styling |
| 14 | Button link hover state with transition |
| 15 | Footer background, color, alignment, and spacing |
